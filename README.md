## Disciplina
**Projeto de Machine Learning (PML)** — IBMEC — **2026.1**  
**Professor:** **Carlos Eduardo Leal de Castro**

Este repositório contém o **template da turma** para iniciar projetos em grupo usando **Cookiecutter**, seguindo uma estrutura de projeto de ML como em empresas: separação clara entre dados, notebooks, código reutilizável, modelos e relatórios.

---

## Objetivo do template
Quando você cria um projeto com este template, você já começa com:

-  **Estrutura padronizada** (igual para todos os grupos)
-  **Pastas e convenções** usadas em times de dados
-  **Código em `src/`** (reutilizável) + **notebooks para exploração**
-  Um caminho natural: **EDA → features → treino → métricas → relatório**
